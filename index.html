TYPO3 Extension st_address_map

With the extension st_address_map you are able to show addresses out of tt_address in Google Maps.

View project on GitHub

What does it do?

With the extension st_address_map you're able to show addresses out of tt_address in Google Maps. Further it's possible to show the addresses in a list. Therefore you're also able to define your own “select fields”. Within a search field the user can search for postal codes. After pushing the searchbutton the direct hits of postal codes and addresses are shown. Please keep in mind, that only the postal codes included in the addresses can be found. Via constant editor for example the searchradius in kilometers can be edited.
